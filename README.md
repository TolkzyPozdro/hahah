# Strona Sakkara

Otwórz `index.html` w przeglądarce.

## Do potwierdzenia przed publikacją

- Dokładny adres restauracji: w materiałach źródłowych pojawia się rozbieżność `Młynarska 1` / `Młynarska 2`.
- Dokładne cytaty opinii z Google Maps: w prototypie są bezpieczne streszczenia i link do wizytówki, ponieważ pełne opinie najlepiej pobrać przez Google Places API albo ręcznie z panelu wizytówki.
- Finalne ceny menu: obecne ceny są przykładowe i powinny zostać zsynchronizowane z aktualnym menu restauracji.

## Zawartość

- Hero z mocnym telefonem i tłem jedzenia.
- Sticky pasek akcji na mobile.
- Popularne dania.
- Menu w zakładkach.
- Promocje.
- Galeria z naturalnymi podpisami.
- Sekcja opinii gotowa pod Google.
- Kontakt z osadzoną mapą Google i formularzem rezerwacji.
